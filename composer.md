# composer

### konfigurasi

<https://getcomposer.org/doc/03-cli.md#environment-variables>


```txt
COMPOSER_HOME=C:\wasp\composer
COMPOSER_CAFILE=C:\wasp\cacert.pem
COMPOSER_BIN_DIR=C:\wasp\bin
COMPOSER_CACHE_DIR=C:\wasp\composer\cache
```

### instalasi

- buka terminal
- cd `./composer`
- lakukan **Command-line installation** berdasarkan https://getcomposer.org/download/
- cek dengan perintah `composer diagnose`
